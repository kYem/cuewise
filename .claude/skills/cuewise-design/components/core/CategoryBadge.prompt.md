Solid color pill that labels a quote's category — sits above the quote, on quote-library cards, and inline (small).

```jsx
<CategoryBadge category="creativity" />
<CategoryBadge category="success" size="sm" />
```

- **category**: `inspiration · learning · productivity · mindfulness · success · creativity · resilience · leadership · health · growth` — each maps to a fixed `--category-*` color.
- **size**: `sm` / `md` / `lg`. Always white text on the category color.
